# Setup Instructions for Metadata Forensic Tool

## Prerequisites
- Ensure you have the following software installed:
  - **Node.js** (version 14 or higher)
  - **npm** (Node Package Manager)
  - **Git** (for version control)

## Installation Steps
1. **Clone the Repository**
   Open your terminal and run:
   ```bash
   git clone https://github.com/JSXVI/metadata-forensic-tool.git
   cd metadata-forensic-tool
   ```

2. **Install Dependencies**
   After navigating to the project directory, install the required npm packages:
   ```bash
   npm install
   ```

## Build Commands
- To build the project, use the following command:
```bash
npm run build
```

## Verification Procedures
1. **Run Tests**
   To verify that everything is functioning properly, run the test cases provided in the repository:
   ```bash
   npm test
   ```

2. **Start the Application**
   After the build is successful, start the application to verify its functionality:
   ```bash
   npm start
   ``` 
   Open your web browser and navigate to `http://localhost:3000` to see if the application is running properly.

3. **Check for Errors**
   Monitor the terminal for any errors during the installation, build, or execution steps. If any issues arise, refer to the error messages for guidance.
